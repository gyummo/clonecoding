구글 계정 연결이 끝나셨나요?
아래 확인 버튼을 눌러 메일 발송 설정에서 연결된 계정을 확인해주세요.

메일 발송 설정 확인

연결창이 실행되지 않았나요? Gmail 연결 을 다시 시도해주세요.

설정방법 알아보기

메일 설정 안내

내 사이트에서 신규가입, 주문 등의 상황에서 자동으로 메일을 보내게 될 때 사이트의 소유자 이메일 주소로 발송하게 됩니다.

하지만 보내는 메일 서버(내 사이트) 와 보내는 메일 주소의 도메인이 다를 경우, 스팸으로 분류되는 현상이 있습니다. 예를 들어, abc@naver.com 이라는 메일 주소로 발송하게 될 때, naver.com 에서 보내는 메일이 아니기 때문에 스팸으로 분류될 가능성이 있습니다.

이렇게 스팸으로 분류되는 현상을 최소화하기 위해서는 아래와 같이 메일 서비스별로 설정이 필요합니다.

  • Gmail
  • 네이버
  • 다음
  • 네이트
  1. 내 사이트의 [사용자 관리 > 메일 발송 설정] 에서 서비스 제공업체를 Gmail 로 선택합니다.

  2. Google 연결 버튼을 클릭합니다.

  3. Google 계정 연결 창이 나오면 연결할 Google 계정을 선택합니다.

  4. '나를 대신하여 이메일 전송' 권한을 선택 허용하고, 계속 버튼을 클릭합니다.

  5. 이 도움말을 닫고, 메일 발송 설정 확인 버튼을 클릭해 메뉴로 돌아갑니다. (어떤 이유로 Google 계정에 대한 액세스 요청 창이 실행되지 않거나 정상적으로 모든 과정을 완료하지 않았다면 위 2번 과정부터 다시 시도해 주세요.)

  6. 메일 발송 설정에서 메일이 잘 연결되었는지 확인합니다.

전송한도: 1회당 최대 100명, 하루 최대 500개

인증메일이 오지 않거나 SMTP 연결 실패시 조치방법

인증메일이 오지 않거나 SMTP 연결 실패 시 아래의 방법을 사용해 조치해 보시길 바랍니다.

  1. Gmail에 접속 후, 로그인 시도 차단 메일을 확인 후 내 활동 확인 버튼을 클릭합니다.
  2. 메일이 오지 않은 경우 Google 계정 설정의 최근 보안 관련 활동에 접속합니다.
  3. 중요 보안 경고 창이 나오면 최근에 Google 계정 로그인이 차단된 적이 있나요? 에서 예 를 클릭합니다.
  4. 메일 발송 설정 에서 메일 인증을 재시도를 실행합니다.

1단계: 네이버 메일 설정

  1. 네이버 메일에 접속합니다. (PC 브라우저에서 접속해 주세요.)
  2. 왼쪽 메뉴 열에서 하단에 위치한 환경설정 을 클릭합니다.
  3. 상단 메뉴에서 POP3/IMAP 설정 을 클릭합니다.
  4. POP3/SMTP 사용 에서 사용함 을 선택합니다.
  5. 저장 버튼을 클릭해 적용합니다.

2단계: 메일 발송 설정

  1. 이 도움말을 닫고, 서비스 제공업체 를 네이버 로 선택합니다.

  2. 아래의 항목들을 설정합니다.

    • 발신자 메일주소: 네이버 메일 주소 입력 (예: myid@naver.com)
    • 로그인 ID: 네이버 ID 입력 (이메일 형식이 아닌 네이버 ID 입력, myid@naver.com)
    • 패스워드: 네이버 비밀번호 입력 (2단계 인증 서비스 이용 시 아래의 별첨 도움말을 참고해 외부 애플리케이션 비밀번호 생성 후 입력)
    • 인증메일 주소: 수신 가능한 메일 주소 입력 (발신자 메일 주소와 같아도 무방)
  3. 인증메일 보내기 버튼을 클릭합니다.
  4. 인증메일 주소로 발송된 인증메일을 열고, 인증번호를 확인합니다.
  5. 인증번호 입력 후, 인증 하기 버튼을 클릭합니다.
  6. 인증이 완료되면 오른쪽 상단에 위치한 저장 버튼을 클릭해 메일 설정을 완료합니다.
  7. 이제 입력하신 발신자 메일 주소로 자동 메일이 발송됩니다.

전송한도: 시간당 최대 30회, 1회당 최대 100명

별첨: 네이버 외부 애플리케이션 비밀번호 생성방법

  1. 네이버 ID에 접속합니다. *네이버 로그인 필수
  2. 기본보안설정 항목에서 2단계 인증 메뉴에 위치한 관리 버튼을 클릭합니다.
  3. 비밀번호를 입력하고, 확인 버튼을 클릭합니다.

  4. 아래의 순서에 따라 새 애플리케이션 비밀번호를 생성합니다.

    1. 종류선택: 직접 입력 을 선택하고, 애플리케이션 이름을 입력합니다. (예: 아임웹 자동메일발송)
    2. 비밀번호 생성: 생성하기 버튼을 클릭합니다.
    3. 비밀번호 확인: 비밀번호가 생성되면 마우스로 드래그하여 복사합니다.
  5. 이제 복사한 외부 애플리케이션 비밀번호를 아임웹에서 네이버 ID 비밀번호로 활용해 보세요.
    참고 : 외부 애플리케이션 비밀번호를 잊어버린 경우, 다시 열어 확인할 수 없으며 새롭게 생성해야 합니다.

1단계: 다음 메일 설정

  1. 다음 메일에 접속합니다. (PC 브라우저에서 접속해 주세요.)
  2. 왼쪽 메뉴 열에서 하단에 위치한 환경설정 을 클릭합니다.
  3. 상단 메뉴에서 IMAP/POP3 를 클릭합니다.
  4. IMAP / SMTP 사용 에서 사용함 을 선택합니다.
  5. 저장을 클릭해 적용합니다.

2단계: 메일 발송 설정

  1. 이 도움말을 닫고, 서비스 제공업체 를 다음 으로 선택합니다.

  2. 아래의 항목들을 설정합니다.

    • 발신자 메일주소: 다음 메일 주소 입력 (예: myid@daum.net, myid@hanmail.net)
    • 로그인 ID: 다음 ID 입력 (이메일 형식이 아닌 다음 ID 입력, myid@daum.net)
    • 패스워드: 다음 ID 비밀번호 입력 (2단계 인증 서비스 이용 시 아래의 별첨 도움말을 참고해, 앱 비밀번호 생성 후 입력)
    • 인증메일 주소: 수신 가능한 메일 주소 입력 (발신자 메일 주소와 같아도 무방)
  3. 인증메일 보내기 버튼을 클릭합니다.
  4. 인증메일 주소로 발송된 인증메일을 열고, 인증번호를 확인합니다.
  5. 인증번호 입력 후, 인증 하기 버튼을 클릭합니다.
  6. 인증이 완료되면 오른쪽 상단에 위치한 저장 버튼을 클릭해 메일 설정을 완료합니다.
  7. 이제 입력하신 발신자 메일 주소로 자동 메일이 발송됩니다.

전송한도: 하루 최대 1만개, 1회당 최대 150개

별첨: 다음 앱 비밀번호 생성방법

  1. 카카오 계정 보안에 접속합니다. *카카오 로그인 필수
  2. 정보 보호 항목에서 2단계 인증 을 클릭합니다.
  3. 비밀번호를 입력하고, 확인 버튼을 클릭합니다.
  4. 앱 비밀번호 오른쪽 끝에 위치한 화살표 아이콘을 클릭합니다.
  5. 앱 비밀번호 이름(예: 아임웹 자동메일발송)를 입력하고, 생성 버튼을 클릭합니다.
  6. 앱 비밀번호 생성 확인 창이 나오면 확인 버튼을 클릭합니다.
  7. 복사하기 버튼를 클릭하면 앱 비밀번호가 복사됩니다.
  8. 이제 복사한 앱 비밀번호를 아임웹에서 다음(Daum) ID 비밀번호로 활용해 보세요.
    참고: 앱 비밀번호를 잊어버린 경우, 다시 열어 확인할 수 없으며 새롭게 생성해야 합니다.

1단계: 네이트 메일 설정

  2. 왼쪽 최하단의 환경설정 을 클릭합니다.
  3. 모바일 관리의 POP3 로 이동합니다.
  4. POP3 사용여부 에서 POP3 사용 을 선택합니다.
  5. 저장 을 클릭합니다.

2단계: 메일 발송 설정

    • 발신자 메일주소: 네이트 메일 주소 입력 (예: myid@nate.com)
    • 로그인 ID: 네이트 ID 입력 (이메일 형식이 아닌 네이트 ID 입력, myid@nate.com)
    • 패스워드: 네이트 ID 비밀번호 입력

  3. 인증메일 보내기 버튼을 클릭합니다.
  4. 인증메일 주소로 발송된 인증메일을 열고, 인증번호를 확인합니다.
